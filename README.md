# 🖥️ desktop-sync

Aplicación de escritorio en Java orientada a **sincronización de archivos y gestión documental**, asociada al paquete `com.openkm.okmsynchronize`.

![Java](https://img.shields.io/badge/Java-desktop-blue?logo=openjdk)

---

## 📌 Objetivo

Proveer una base de sincronización entre entorno local y un sistema documental, con componentes de interfaz de escritorio y controladores dedicados.

## 🧩 Qué incluye

- Código fuente Java bajo `src/com/openkm/okmsynchronize/`.
- Controladores para configuración, alertas y sincronización.
- Librerías externas en `lib/`.
- Configuración de proyecto estilo Eclipse/NetBeans.

## ▶️ Uso general

```bash
git clone git@github.com:Carl0gonzalez/desktop-sync.git
cd desktop-sync
```

Compila el proyecto en tu IDE Java o ajusta un script de compilación según las dependencias incluidas en `lib/`.

## 💡 Valor del proyecto

Es una pieza interesante para mostrar integración Java desktop con procesos documentales o de sincronización empresarial.

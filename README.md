# 🖥️ desktop-sync

Aplicación de escritorio en Java asociada al paquete `com.openkm.okmsynchronize`, orientada a **sincronización documental y gestión de archivos**.

![Java](https://img.shields.io/badge/Java-desktop-blue?logo=openjdk)

## Descripción

El repositorio contiene controladores, vistas y librerías externas para una solución desktop que parece interactuar con OpenKM u otro sistema documental. Su estructura corresponde a un proyecto Java clásico de escritorio.

## Objetivo

Mantener una base funcional para explorar sincronización documental, integración de escritorio y automatización de tareas de gestión de archivos.

## Uso

Abre el proyecto en tu IDE Java y ajusta dependencias o scripts según el entorno local.
